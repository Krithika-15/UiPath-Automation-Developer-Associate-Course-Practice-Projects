# UiPath Automation Developer Associate – Practice Projects

This repository contains UiPath automation workflows created while learning the **UiPath Automation Developer Associate course**.  
The projects focus on practicing core RPA concepts such as **data manipulation, string operations, selectors, UI automation, and workflow design**.

Each project demonstrates a specific concept learned during the course through small hands-on implementations.

- UiPath Studio (Modern Design Experience)
- .NET String Methods
- UiPath Data Manipulation Activities
- Logging and Debugging
- Workflow Modularization
- UI Automation Concepts

---

# 📂 Projects

---

## 1️⃣ Email Address Extraction Bot

This project extracts an email address from a given sentence using **string manipulation methods**.

Two different approaches are implemented to demonstrate multiple ways of solving the same problem.

### Concepts Used

- `IndexOf`
- `Substring`
- String length calculation
- Assign activity
- Conditional logic
- Log Message activity
- Modular workflows using **Invoke Workflow**

### Learning Outcome

Understanding how **string operations can be used to extract structured information from unstructured text**.

---

## 2️⃣ String Bookstore Demo

This project demonstrates different **string manipulation techniques** to extract and process information related to bookstore availability.

The automation processes text data, extracts author information, and formats bookstore availability using string operations.

### Concepts Used

- `Split`
- `Substring`
- `LastIndexOf`
- `String.Format`
- `String.Join`
- List operations
- Data formatting

### Learning Outcome

Learning how **text data can be parsed, transformed, and formatted using .NET string methods in UiPath workflows**.
